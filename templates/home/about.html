About

PrivMsg is a Free and Open Source Software that is also self-hostable. It is built to be an instant messaging and burn note service. It is written and built with django channels and mongodb.
PrivMsg uses AES-256 bit encryption to encrypt your messages and notes. Every burn note link comes with two parts - the identifier and a password which is separated by a semi colon. PrivMsg only stores the identifier, while the password which is used to encrypt the link, it is never stored in the database.
